Functions

  • Function Declaration

    • A function is an independent block of code that performs a specific task.

  • Function Expression

    • A function expression is a way to store functions in variables.

  • Arrow Function

    • Arrow function {()=>} is concise way of writing JavaScript functions in shorter way. Arrow functions were introduced in the ES6 version. They make our code more structured and readable. Arrow functions are anonymous functions i.e. functions without a name but they are often assigned to any variable. They are also called Lambda Functions. Syntax: const person = () => { console.log( "raghu" ); }
