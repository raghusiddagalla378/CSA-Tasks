Datatypes in JavaScript

  1. primitive datatypes:

    2. They can hold a single simple value.

    • Strings
    • Numbers
    • BigInt
    • Boolean
    • Undefined
    • null
    • Symbol
  2. Non-Primitive datatypes:

    3. They can hold multiple values.

    • Objects

      • An Object holds data in the form of key-value pairs.

    • Arrays
    • RegExpr

typeof():

The typeof operator allows you to check the type of the variable.

Array

array is a data structure in javascript that allows you to store the multiple values in a single variables.
